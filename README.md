# Curl & Postman

Let's start by running some commands both in cURL and Postman using a free public API.  Here are a options:

### APIs to Use  
- [JSONPlaceholder](https://jsonplaceholder.typicode.com/) 
- [Cat Facts API](https://catfact.ninja/fact) 
- [Advice Slip API](https://api.adviceslip.com/) 
- [The Dog API](https://thedogapi.com/) 
- [Bored API](https://www.boredapi.com/api/activity) 
- [ReqRes](https://reqres.in/)

### Objectives
Choose one to run some cURL commands in.  Follow the API's documentation to learn more about what you can and can't do with the API.

As you write out your cURL commands, save them to the curl.md document in the solutions folder of this repository.

Then do the same with Postman, except with a **different** API.  And instead of copy/pasting your queries, you'll be exporting them into a single file.  The easiest way to do this is to save all of your requests into a single collection which you can then export into a json.  Throw that in the solutions folder as well.
